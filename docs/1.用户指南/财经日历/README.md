## 功能概述

财经日历数据处理系统提供以下核心功能：

1.  通过工作流脚本处理历史和实时财经事件数据。
2.  基于 YAML 文件进行高度可配置的事件筛选（重要性、关键词、货币、时间等）。
3.  自动添加市场开盘事件（可选）。
4.  将处理结果导出为 CSV, JSON, 或 MT5 兼容格式。
5.  与 MT5 集成，自动复制文件到 MT5 目录（可选）。

通过本目录及相关资源中的文档，您可以全面了解如何配置和运行本系统，获取用于市场分析和交易决策的财经事件数据。
